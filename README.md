# Entity Framework Core(EF_core)

EF Core is a lightweight, cross-platform ORM framework that enables .NET applications to interact with databases using C# objects instead of writing SQL manually.

# What is ORM?

ORM (Object Relational Mapping) maps database tables to C# classes and database rows to objects.

# What are the main benefits of EF Core?
  Less SQL coding
  Faster development
  LINQ support
  Migrations
  Change tracking
  Database independence
  
# What are the limitations of EF Core?
  Performance overhead
  Complex query optimization issues
  Less control over generated SQL
